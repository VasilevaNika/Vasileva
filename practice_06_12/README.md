Задача Конверт валют:
1) Поддерживаемые валюты: USD, EUR, RUB, GBR
2) Используется словарь для хранения курсов валют (Все курсы заданы относительно USD)
3) Конвертация из одной валюты в другую
4) История операций

Запуск:
программа отображает доступные валюты
нужно ввести валюту, из которой хотите конвертировать
ввести валюту, в которую хотите конвертировать
ввести сумму для конвертации
чтобы просмотреть историю надо ввести "история"
чтобы выйти из программы ввести "выход"


Пример:
Валюты: USD, EUR, RUB, GBP
'выход' или 'история'

Из: USD
В: EUR
Сумма: 100
Результат: 100.00 USD = 86.00 EUR (курс: 0.8600)
